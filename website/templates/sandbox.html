Quick Start

Try these classic circuits:

  • Bell state: H(q0) → CNOT(q0, q1) — creates maximum entanglement
  • GHZ state: H(q0) → CNOT(q0, q1) → CNOT(q1, q2) — 3-qubit entanglement
  • Superposition: H on every qubit — equal probability of all outcomes
  • Bit flip: X(q0) → CNOT(q0, q1) — copies qubit 0 to qubit 1

Gate Reference

Single-qubit gates:
  H    Hadamard: creates superposition
  X    Pauli-X (NOT): bit flip |0⟩↔|1⟩
  Y    Pauli-Y: bit + phase flip
  Z    Pauli-Z: phase flip |1⟩→−|1⟩
  S    Phase gate: |1⟩→i|1⟩ (√Z)
  T    π/8 gate: |1⟩→e^(iπ/4)|1⟩ (⁴√Z)
  RX   X rotation by π/4
  RY   Y rotation by π/4
  RZ   Z rotation by π/4

Two-qubit gates:
  CNOT  Controlled-NOT (flips target if control is |1⟩)
  CZ    Controlled-Z (phase flip if both |1⟩)
  SWAP  Swaps two qubits
