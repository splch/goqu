Quantum Gates as Matrices

Quantum gates are unitary matrices that transform qubit states. Being unitary means they preserve probability: if |α|² + |β|² = 1 before the gate, it remains 1 after.

Pauli Gates

The three Pauli gates are fundamental single-qubit operations:

X = [0 1]    Y = [0 -i]    Z = [1  0]
    [1 0]        [i  0]        [0 -1]

X|0⟩ = |1⟩  (bit flip)
Y|0⟩ = i|1⟩ (bit + phase flip)
Z|0⟩ = |0⟩  (phase flip — no effect on |0⟩)

Hadamard Gate

H = 1/√2 [1  1]
          [1 -1]

H|0⟩ = |+⟩ = (|0⟩ + |1⟩)/√2
H|1⟩ = |−⟩ = (|0⟩ − |1⟩)/√2

Phase Gates

S = [1 0]    T = [1    0  ]
    [0 i]        [0 e^iπ/4]

S is a quarter-turn, T is an eighth-turn around the Z axis.

The Bloch Sphere

Any single-qubit state can be visualized as a point on the Bloch sphere. The north pole is |0⟩, south pole is |1⟩, and the equator contains superposition states like |+⟩ and |−⟩.

  • X gate: 180° rotation around the X axis (flips |0⟩ ↔ |1⟩)
  • Z gate: 180° rotation around the Z axis (phase flip)
  • H gate: 180° rotation around the X+Z axis
  • RX(θ), RY(θ), RZ(θ): Rotation by angle θ around the corresponding axis

Goqu Code

// Apply various gates
c, _ := builder.New("gates", 1).
    H(0).           // Hadamard
    RZ(math.Pi/4, 0). // Z-rotation by π/4
    H(0).           // Back through Hadamard
    MeasureAll().
    Build()

fmt.Println(draw.String(c))
// q0: -H---RZ(pi/4)---H---M--

Exercises

  1. Try applying the S and T gates. What happens to the measurement probabilities? Why?
  2. The S gate satisfies S² = Z. Verify this by thinking about the matrices.
  3. What gate sequence would create the state (|0⟩ + i|1⟩)/√2?
