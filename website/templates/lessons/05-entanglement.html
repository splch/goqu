5. Entanglement & Bell States

{{.Lesson.Desc}}

Learning Objectives

  • Understand quantum entanglement and why it's important
  • Learn the four Bell states and how to create them
  • See entangled correlations in measurement results
  • Explore the CNOT gate's role in creating entanglement

What is Entanglement?

Two qubits are entangled when their joint state cannot be written as a product of individual states. For example:

|Φ⁺⟩ = (|00⟩ + |11⟩) / √2

This state cannot be factored as |ψ₁⟩ ⊗ |ψ₂⟩. If you measure one qubit and get 0, the other is guaranteed to be 0 too. If you get 1, the other is also 1. This correlation holds even if the qubits are separated by vast distances.

The CNOT Gate

The Controlled-NOT (CNOT) is the key two-qubit gate. It flips the target qubit if and only if the control qubit is |1⟩:

CNOT|00⟩ = |00⟩    CNOT|10⟩ = |11⟩
CNOT|01⟩ = |01⟩    CNOT|11⟩ = |10⟩

The Four Bell States

The Bell states are the four maximally entangled two-qubit states:

|Φ⁺⟩ = (|00⟩ + |11⟩) / √2    ← H(q0), CNOT(q0,q1)
|Φ⁻⟩ = (|00⟩ − |11⟩) / √2    ← X(q0), H(q0), CNOT(q0,q1)
|Ψ⁺⟩ = (|01⟩ + |10⟩) / √2    ← H(q0), CNOT(q0,q1), X(q1)
|Ψ⁻⟩ = (|01⟩ − |10⟩) / √2    ← X(q0), H(q0), CNOT(q0,q1), X(q1)

Notice how the measurement results are always correlated: for |Φ⁺⟩, you only see |00⟩ and |11⟩, never |01⟩ or |10⟩. This is the hallmark of entanglement.

Goqu Code

// Create a Bell state
c, _ := builder.New("bell", 2).
    H(0).
    CNOT(0, 1).
    MeasureAll().
    Build()

sim := statevector.New(2)
counts, _ := sim.Run(c, 1024)
// counts ≈ {"00": 512, "11": 512}
