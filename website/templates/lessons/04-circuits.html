The Circuit Model

Quantum computation is expressed as a sequence of quantum gates applied to qubits, forming a circuit. Circuit diagrams read left to right:

  • Horizontal lines represent qubits (wires)
  • Boxes on wires represent gates
  • Lines connecting wires represent multi-qubit gates
  • A meter symbol (M) represents measurement

Multi-Qubit States

With two qubits, we have four basis states: |00⟩, |01⟩, |10⟩, |11⟩. The general state is:

|ψ⟩ = α₀₀|00⟩ + α₀₁|01⟩ + α₁₀|10⟩ + α₁₁|11⟩

With n qubits, we have 2n basis states — this exponential growth is key to quantum computing's power.

Goqu Code

// Build a 2-qubit circuit
c, _ := builder.New("my-circuit", 2).
    H(0).        // Hadamard on qubit 0
    CNOT(0, 1).  // CNOT with control=0, target=1
    MeasureAll().
    Build()

// Visualize the circuit
fmt.Println(draw.String(c))
// q0: -H---@---M--
//          |
// q1: -----X---M--

// Get the SVG for web display
svg := draw.SVG(c)

// Simulate
sim := statevector.New(2)
counts, _ := sim.Run(c, 1024)

Exercises

  1. What output do you expect from X(q0) → CNOT(q0,q1)? Try it and verify.
  2. Try H(both) → CNOT → H(both). This is equivalent to a CNOT with reversed control/target. Can you see why?
  3. How many complex amplitudes are needed to describe the state of 10 qubits? 20 qubits? 50 qubits?
