What is a Qubit?

A qubit (quantum bit) is the quantum analogue of a classical bit. Like a classical bit, it has two basis states: |0⟩ and |1⟩. But unlike a classical bit, a qubit can exist in a superposition of both states simultaneously.

The general state of a qubit is:

|ψ⟩ = α|0⟩ + β|1⟩

where α and β are complex numbers called probability amplitudes, satisfying |α|² + |β|² = 1.

Measurement

When we measure a qubit, the superposition collapses:

  • We get outcome 0 with probability |α|²
  • We get outcome 1 with probability |β|²

After measurement, the qubit is in the measured state — the superposition is destroyed. This is irreversible.

Dirac Notation

We use Dirac notation to describe quantum states:

  • |0⟩ = the "zero" state (column vector [1, 0]ᵀ)
  • |1⟩ = the "one" state (column vector [0, 1]ᵀ)
  • |+⟩ = (|0⟩ + |1⟩) / √2 — equal superposition
  • |−⟩ = (|0⟩ − |1⟩) / √2 — equal superposition with relative phase

The Hadamard gate (H) creates an equal superposition: H|0⟩ = |+⟩ = (|0⟩ + |1⟩)/√2. When you measure, you get |0⟩ or |1⟩ with roughly equal probability.

Goqu Code

// Create a qubit in superposition and measure it
c, _ := builder.New("superposition", 1).
    H(0).          // Apply Hadamard to qubit 0
    MeasureAll().
    Build()

sim := statevector.New(1)
counts, _ := sim.Run(c, 1024)
// counts ≈ {"0": 512, "1": 512}

Exercises

  2. The Pauli-Z gate: what happens when you measure after applying Z to |0⟩? Can you explain why?
  3. If a qubit has state |ψ⟩ = (√3/2)|0⟩ + (1/2)|1⟩, what is the probability of measuring |0⟩? |1⟩?
